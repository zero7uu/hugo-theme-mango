# 图片展示示例

本文档展示了如何使用主题提供的图片展示功能。

## 1. 单张配图

使用 `{{</* image */>}}` 短代码来展示单张配图。

### 基本用法

```
{{</* image src="https://example.com/image.jpg" alt="示例图片" */>}}
```

### 带图注的用法

```markdown
{{</* image src="https://example.com/image.jpg" alt="示例图片" caption="测试图注功能" */>}}
```

## 2. 响应式设计

这些图片展示功能都支持响应式设计：
- 在大屏幕上，图片会居中显示，宽度自适应容器
- 在手机等小屏幕设备上，所有图片都会自动调整为100%宽度显示

这样，无论你的读者使用什么设备访问你的网站，都能获得良好的图片浏览体验。

## 3. 使用说明

### 语法格式

```markdown
{{</* image src="图片URL" alt="图片描述" caption="图注文字（可选）" */>}}
```

### 参数说明

- `src`：图片的URL地址
- `alt`：图片的替代文字，用于无障碍访问
- `caption`：图片的图注文字，显示在图片下方

### 注意事项

1. 图片URL必须是完整的，包括协议（http://或https://）
2. 图片描述和图注是可选的，但建议添加图片描述以提高无障碍访问性
3. 图片会自动适应容器宽度，不需要担心图片尺寸过大的问题
